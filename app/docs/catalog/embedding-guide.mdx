## Handling Cross-Origin Issues

Vizualni Admin uses iframe-resizer to handle cross-origin communication. Include the necessary scripts on your embedding page:

```html
<!DOCTYPE html>
<html>
<head>
  <script src="https://cdn.jsdelivr.net/npm/iframe-resizer@4/js/iframeResizer.min.js"></script>
</head>
<body>
  <iframe
    id="vizualni-admin-chart"
    src="https://your-domain.com/embed/chart-123"
    width="100%"
    height="500"
    frameborder="0"
  ></iframe>
  
  <script>
    iFrameResize({ log: false }, '#vizualni-admin-chart');
  </script>
</body>
</html>
```

This ensures proper resizing and communication between the parent page and the embedded chart.
