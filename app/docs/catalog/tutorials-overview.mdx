# Tutorials Overview

<ImageSpecimen src="./static/docs/tutorials-hero.png" plain />

Vizualni Admin offers a comprehensive tutorial system designed to help users of all skill levels learn how to create, customize, and embed data visualizations using Serbian open data from data.gov.rs. This overview explains the tutorial structure, navigation, and how to get the most out of the learning resources.

## What Tutorials Are Available

The tutorial system is organized into five main categories, each containing step-by-step guides with interactive examples and practical exercises:

### Getting Started
- **Your First Visualization**: Learn the basics of creating a simple chart from data.gov.rs datasets
- **Exploring the Interface**: Tour the main features and tools available in Vizualni Admin

### Creating Charts
- **Understanding Chart Types**: Overview of available visualizations (bar, line, pie, map, etc.)
- **Advanced Chart Customization**: Styling, colors, and interactive features
- **Working with Data**: Data preparation, filtering, and transformations

### Embedding
- **Basic Embedding**: How to embed visualizations on external websites
- **Embedding in WordPress**: Step-by-step guide for WordPress integration
- **Responsive Embedding**: Making embeds work on all device sizes

### API Usage
- **Using data.gov.rs API**: Finding and using Serbian open data
- **API Best Practices**: Rate limits, error handling, and optimization
- **Custom Data Sources**: Integrating external APIs and data formats

### Advanced Topics
- **Custom Data Transformations**: Advanced data manipulation techniques
- **Performance Optimization**: Improving load times and user experience
- **Contributing to Vizualni Admin**: How to extend and customize the tool

## How to Navigate Tutorials

Tutorials are accessible through the main navigation menu under "Tutorials" or directly at `/tutorials`. The tutorials index page displays all available tutorials in a searchable grid.

<ImageSpecimen src="./static/docs/tutorials-index.png" plain />

Each tutorial page features:
- A progress indicator showing your current step
- Navigation buttons for moving between steps
- Interactive code examples with copy-to-clipboard functionality
- Live demo previews where applicable
- A sidebar table of contents for quick navigation

## How Tutorials Complement Demos

While the [demo gallery](/demos) showcases finished visualizations across different categories, tutorials provide guided learning experiences:

- **Demos**: Static examples for inspiration and reference
- **Tutorials**: Interactive learning with step-by-step instructions

Tutorials often reference specific demos as examples, creating a seamless learning flow from theory to practice.

## Recommended Learning Paths

### For Beginners
1. Start with "Your First Visualization" (Getting Started)
2. Explore "Understanding Chart Types" (Creating Charts)
3. Try "Basic Embedding" (Embedding)

### For Developers
1. Complete the beginner path above
2. Learn "Using data.gov.rs API" (API Usage)
3. Dive into "Custom Data Transformations" (Advanced Topics)

### For Data Analysts
1. Focus on "Working with Data" (Creating Charts)
2. Study "Advanced Chart Customization" (Creating Charts)
3. Explore "API Best Practices" (API Usage)
