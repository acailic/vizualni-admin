> This pattern library provides components for designing clean and accessible data visualizations.

## Principles

The goal is to establish a visual coherence for data.gov.rs visualizations and provide accessible, user-friendly components.

Key design elements:
- Website header with data.gov.rs branding
- Clean layout separating header from content
- Footer with copyright and legal information
- «Noto Sans» typeface for readability
- Accessible color palette

## Design Approach

Visualize serves specific data visualization use cases with unique requirements. Our design system focuses on:

- Creating accessible and readable charts and maps
- Using an extended color range optimized for data visualization
- Maintaining consistency across the data.gov.rs platform
