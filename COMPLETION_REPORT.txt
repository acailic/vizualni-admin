================================================================================
                          USER NEXT STEPS
================================================================================

To use this repository:

1. Install dependencies:
   $ yarn install

2. Start database:
   $ docker-compose up -d

3. Configure environment:
   $ cp .env.example .env.local
   $ # Edit .env.local with your settings

4. Run migrations:
   $ yarn db:migrate:dev

5. Compile translations:
   $ yarn locales:compile

6. Start development server:
   $ yarn dev

7. Open browser:
   http://localhost:3000
